## Registered Agent

### Key Registry Fields

| Field | Purpose |
|-------|---------|
| `registry.catalog_id` | Unique identifier in the agent catalog |
| `registry.catalog_classification` | Domain, subdomain, and capability taxonomy |
| `registry.visibility` | Access scope (`internal`, `public`, `restricted`) |
| `registry.federation` | Cross-registry discovery and sync configuration |
