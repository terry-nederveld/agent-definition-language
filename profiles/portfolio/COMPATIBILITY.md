# Portfolio Profile Compatibility

This document tracks compatibility between Portfolio Profile versions and ADL specification versions.

## Compatibility Matrix

| Profile Version | ADL Versions | Status | Notes |
|-----------------|--------------|--------|-------|
| 1.0             | 0.1.x        | Draft  | Initial release |

## Compatibility Policy

- **Minor ADL updates** (e.g., 0.1.0 → 0.1.1): Profile remains compatible
- **Major ADL updates** (e.g., 0.1.x → 1.0.0): Profile compatibility will be evaluated; new profile version released if needed
- **Breaking profile changes**: Require new major profile version
